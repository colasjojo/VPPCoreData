NSManagedObject(VPPCDActiveRecord) Category Reference

Conforms to VPPCoreDataActiveRecord
Declared in NSManagedObject+VPPCDActiveRecord.h
NSManagedObject+VPPCDActiveRecord.m

Overview

This category adds ability to perform Active Record actions from different managed object context with a simple syntax.

For example:

NSManagedObjectContext *aDifferentMOC = … ; [[Quote moc:aDifferentMOC] all]; [[Quote moc:aDifferentMOC] findBy:somePredicate];
